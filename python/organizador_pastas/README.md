# 📁 Organizador de Pastas

Organiza automaticamente qualquer pasta bagunçada em subpastas por tipo de arquivo.

## Como usar

```bash
# Organiza a pasta Downloads
python organizador.py

# Organiza outra pasta
python organizador.py --pasta /caminho/da/pasta

# Só mostra o que faria, sem mover nada
python organizador.py --simular
```

## Categorias criadas

Imagens · Vídeos · Áudio · Documentos · Planilhas · Apresentações · Código · Compactados · Outros

## Tecnologias
- Python puro — sem nenhuma instalação (`shutil`, `pathlib`, `argparse`)
