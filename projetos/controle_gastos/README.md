# 💰 Controle de Gastos

Registra despesas pelo terminal, agrupa por categoria e gera gráfico de barras. Tudo salvo em CSV entre sessões.

## Como usar

```bash
pip install pandas matplotlib
python gastos.py
```

## Tecnologias
- **pandas** — salvar/carregar CSV, agrupar por categoria
- **matplotlib** — gráfico de barras
